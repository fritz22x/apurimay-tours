# ApuRimay Tours

Sitio catalogo de tours construido con Next.js.

## Scripts

- `npm run dev`: inicia el entorno local.
- `npm run build`: genera la version de produccion.
- `npm run lint`: ejecuta verificacion de TypeScript.

## Estructura

- `app/`: rutas y paginas.
- `src/data/`: datos del negocio y tours.
- `src/components/`: componentes de layout y tarjetas.
- `src/lib/`: utilidades como enlaces de WhatsApp.
